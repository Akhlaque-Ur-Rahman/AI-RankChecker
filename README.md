# AI-RankChecker

RankCheck Pro — Next.js UI (`web/`) + Google Apps Script (`web/apps-script/`) for SERP rank checks via Serper.

## Run locally

```bash
cd web
npm install
npm run dev
```

## Deploy

Set Vercel **Root Directory** to `web`. Configure Google Apps Script Web App (Anyone) and paste the `/exec` URL in the app.
